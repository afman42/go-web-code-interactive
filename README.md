# CODE INTERACTIVE WEB

### Kebutuhan

- Install NodeJS terbaru
- Install Golang terbaru

### Development

- Menjalankan `go run .` atau `go run *.go`

### Production

- Menjalankan `go build`

### Kendala

- PATH file NodeJS executable di windows ketika production
